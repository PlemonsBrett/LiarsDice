C++23 Standards
===============

The project uses modern C++23 features and follows industry best practices.

Naming Conventions
------------------

- Classes: ``PascalCase``
- Functions: ``snake_case``
- Variables: ``snake_case``
- Constants: ``kConstantName``

Code Formatting
===============

Use clang-format with the provided configuration.

   ./scripts/format.sh
